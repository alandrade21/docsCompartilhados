Verifique qual a placa instalada:

$ lspci -vnn | grep -i VGA -A 12


Vai aparecer algo como:

01:00.0 VGA compatible controller [0300]: NVIDIA Corporation G80 [GeForce 8800 GTS] [10de:0193] (rev a2)

Alternativamente instale o pacote nvidia-detect. Esse pacote indica o driver a ser instalado.

Verifique o suporte opengl com 
$ glxinfo | grep OpenGL | grep renderer

Instale o driver

$ sudo apt-get install nvidia-driver

Também é necessário instalar:

libgl1-nvidia-glx:i386

Pode ser que seja necessário habilitar suporte à plataforma 32 bits:

dpkg --add-architecture i386
apt-get update

Configure o servidor X para usar o driver nvidia. Instale o pacote nvidia-xconfig e rode o app. Essa configuração aparece no arquivo /etc/X11/xorg.conf. Exemplo abaixo:

Section "Device"
        Identifier "My GPU"
        Driver "nvidia"
EndSection

A configuração abaixo é indicada para performance:

Section "Device"
        Identifier "My GPU"
        Driver "nvidia"
        Option "NoLogo" "1"
        Option "RenderAccel" "1"
        Option "TripleBuffer" "true"
        Option "MigrationHeuristic" "greedy"
EndSection

Use nvidia-settings para configurar outros parâmetros.
