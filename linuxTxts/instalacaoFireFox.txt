A partir do debian 9 o firefox já é o navegador padrão. Não é necessário fazer mais nada.


Adicionar novo repositório no arquivo /etc/apt/sources.list

deb http://packages.linuxmint.com debian import

Adicionar a chave pública desse repositório

# apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 3EE67F3D0FF405B2

Update package list

# apt-get update

Instale o firefox

# apt-get install firefox
