$ sudo iotop

O tracker é um indexador de buscas que consome muito recurso do sistema, para desabilitá-lo globalmente:

 - vá para /etc/xdg/autostart/ e edite todos os arquivos .desktop que contenham tracker
 - adicione Hidden=true no final do arquivo.

Para desabiblitá-lo em um usuário:

 - vá para o diretório ~/.config/autostart (ou crie-o caso não exista)
 - crie um arquivo .desktop para os arquivos .desktop citadoa acima aue vc quer desligar.
 - O conteúdo desse sarquivos deve ser o seguinte:

[Desktop Entry]
Encoding=UTF-8
Name=Tracker
Hidden=true
