- Proxy da instalação fica /etc/apt/apt.conf
- Wget
   - Arquivo /etc/wgetrc
   - use_proxy = on
   - http_proxy = http://usuario:senha@proxy:porta/
   - https_proxy = http://usuario:senha@proxy:porta/
   - ftp_proxy = http://usuario:senha@proxy:porta/
- Profile
   - Arquivo .profile na raiz do home
   - export http_proxy="http://usuario:senha@proxy:porta/"
   - export https_proxy="http://usuario:senha@proxy:porta/"
   - export ftp_proxy="http://usuario:senha@proxy:porta/"
- NPM
   - Arquivo /root/.npmrc
   - proxy = http://usuario:senha@proxy:porta/
   - https-proxy = http://usuario:senha@proxy:porta/
   
   - Arquivo .npmrc na raiz do home
   - proxy = http://usuario:senha@proxy:porta/
   - https-proxy = http://usuario:senha@proxy:porta/
