Find number of sound cards present in your machine

# cat /proc/asound/cards

How to get card manufacture details using lspci command.

# lspci -v | grep -i audio

 ou 

# alsamixer

Para redirecionar o mic para a saida

# pactl load-module module-loopback latency_msec=1

Retorna um número. Para parar o redirecionamento faz

# pactl unload-module n

Onde n é o número retornado pelo comando de load.
