- Instalar node JS
  ----------------

$ curl -sL https://deb.nodesource.com/setup_11.x | sudo -E bash -
$ sudo apt-get install nodejs
$ sudo apt-get install build-essential

- Testa a instalação

$ node -v
$ npm -v

- Instalar Dependência como DEV
  -----------------------------

$ npm install <pacote> -D
ou
$ npm install @types/node --save-dev
